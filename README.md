# 🎮 Tic-Tac-Toe

A modern and interactive Tic-Tac-Toe game built with **HTML**, **CSS**, and **JavaScript**. Play against your friend or challenge the AI with three different difficulty levels.

## ✨ Features

- 👥 Two Player Mode
- 🤖 AI Opponent
  - 🟢 Easy
  - 🟡 Medium
  - 🔴 Hard
- ✅ Win Detection
- 🤝 Draw Detection
- ❌ Lose Detection
- 🔊 Sound Effects
- 🔄 Dynamic Turn Messages
- 🔁 Restart Game
- 🎨 Smooth UI

## 🛠 Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/fady-albert/TIC-TAC-TOE.git
   ```

2. Navigate to the project folder:
   ```bash
   cd TIC-TAC-TOE
   ```

3. Extract the `sounds.zip` file into the project directory.

4. Open `index.html` in your preferred web browser.
